## What this domain covers

Algorithmic underwriting and pro-forma modeling, an automated valuation model, GIS site selection, and a data-driven developer product. Each build applies the field, ships a runnable artifact with a concrete self-check, and proves the skill with a teach-back.

**What it isn't.** A degree or credential. A claim past what each build's self-check proves. Not medical, legal, or financial advice.
